# IAteNoodles Portfolio

Static portfolio site for Abhijit Kumar Singh, aka IAteNoodles.

## Deploy on GitHub Pages

This site is plain HTML and CSS. It does not use SSR, React, Next.js, Vite, or any build step.

### Option 1: User site

1. Create a repository named `iatenoodles.github.io`.
2. Upload `index.html` to the repository root.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save. Your site will be available at `https://iatenoodles.github.io/`.

### Option 2: Project site

1. Create any repository, for example `portfolio`.
2. Upload `index.html` to the repository root.
3. Go to Settings → Pages.
4. Set Source to `Deploy from a branch`.
5. Choose branch `main` and folder `/root`.
6. Save. Your site will be available at `https://iatenoodles.github.io/portfolio/`.

## Files

- `index.html` — complete static portfolio page
- `README.md` — deployment instructions
- `.nojekyll` — disables Jekyll processing on GitHub Pages
